The libraries in this directory are NOT necessary for building the RI.

However, they are needed if you want to build some of the examples.
They are not necessarily the latest versions -- please get them from the
appropriate locations if you want to get the latest versions.

jakarta-regexp - Used under the Apache software license (included).  This is ONLY RELEVANT for examples.

log4j - used under the Apache Software license (included). The logging
  	support will be changed to used log4j but currently just writes to a 
	file.

junit - is used under the common public license.
	 http://www.opensource.org/licenses/cpl.php

sipxcommons.jar - this is used for generation of nice log files that are
viewable using the sipx trace viewer tool.
