Tests for whether the DialogTimedOutEvent is delivered to the application under the following scenarios:
Scenario 1
  • INVITE is sent by UAC and forked through a forking proxy to a pair of UAS.
  • One UAS returns OK. The UAC ACKs the OK and sends BYE.
  • Another UAS returns OK. The UAC does not ACK the OK.

Scenario 2
  • Inivte sent by UAC through forking proxy to two target UAS
  • One UAS returns RINGING and then OK
  • Another returns RINGING but no OK
